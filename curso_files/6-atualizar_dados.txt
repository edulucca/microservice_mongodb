Atualizar dados e documentos.

1 - Atualizar um dado - Sintaxe: db.<collection>.updateOne({campo_alvo: valor_alvo}, {$set: {campo: novo valor}});

	1.1 - Atualizar a partir do id: db.users.updateOne({_id:ObjectId('6992532787b242aca27c290a')}, {$set:{highscore: 600}});
	1.2 - Atualizar a partir de um campo: db.users.updateOne({name: "Daniel Ferreira"}, {$set: {highscore: 750}});
	1.3 - Atualizar múltiplos dados: db.users.updateOne({name: "Daniel Ferreira"}, {$set: {highscore: 750, hp: 90}});
	1.4 - Observação: se o 'WHERE' do update encontrar mais de uma 'linha', será atualizado apenas uma, por ser updateOne.

2 - Atualizar várias linhas - Sintaxe: db.<collection>.updateMany({campo_alvo: valor_alvo}, {$set: {campo: novo valor}});
	
	2.1 - Atualizar várias linhas: db.users.updateMany({game: "My Game"}, {$set: {money: 200}});
	2.2 - Atualizar todos os documentos: db.users.updateMany({}, {$set:{item:"Espada"}})
		2.2.1 - Observação: o campo 'item' não existia antes do update. Por padrão, o MongoDB cria o campo caso ele não exista em uma operação de update.

3 - É possível criar uma variável para guardar o ObjectId e posteriormente usar o id no update:

	> var id = ObjectId("6992532787b242aca27c290a");
	> db.users.updateOne({_id:id}, {$set:{highscore: 610}});
